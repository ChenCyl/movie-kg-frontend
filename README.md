# movie-kg-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

## Feature

### 搜索

- 模糊搜索
  - 电影名 / 人名

- 豆瓣高分（ >= 9 ）电影

- 拍电影最多的演员

### 交互
- 双击节点搜索

- 自定义样式
  - 节点大小
  - 连线宽度
  - 连线长度
  - 作用力
  - 黑暗/明亮模式

- 搜索结果列表和图的映射

## Vue + D3

写了一个 D3 力导向图的 Vue 组件。更多细节：[vue-network-d3](https://github.com/ChenCyl/vue-network-d3)
